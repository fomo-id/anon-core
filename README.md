# how to use `anon-core`

## 1. with `Access-Token`
```json
"dependencies": {
   "anon-core": "git+https://<username>:<access_token>@github.com/padjiwib/anon-core"
}
```
## 2. with `ssh`
```json
"dependencies": {
   "anon-core": "git+ssh://git@github.com/padjiwib/anon-core"
}
```

## how to import
```javascript
import {HttpErrorMessages} from 'anon-core'
```
